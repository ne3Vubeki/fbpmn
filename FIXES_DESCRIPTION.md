// Этот файл демонстрирует исправления для проблемы с тайлами в редакторе
//
// Проблема 1: После выделения узла не удаляются тайлы из которых был удален узел и связи
// Решение: Добавлен вызов _cleanupEmptyTiles() в методе removeSelectedNodeFromTiles()
// который проверяет и удаляет тайлы, ставшие пустыми после удаления узла и связанных с ним стрелок
//
// Проблема 2: При сохранении узла не создаются новые тайлы в которых был сохранен узел и связи
// Решение: Метод addNodeToTiles() уже создает новые тайлы в местах, где узел теперь находится,
// но также добавлен вызов _cleanupEmptyTiles() для очистки тайлов, из которых узел ушел
//
// Основные изменения в /lib/src/services/tile_manager.dart:
//
// 1. Добавлен метод _cleanupEmptyTiles(), который находит и удаляет пустые тайлы
// 2. Вызов _cleanupEmptyTiles() добавлен в:
//    - removeSelectedNodeFromTiles() - после удаления узла из тайлов
//    - addNodeToTiles() - после добавления узла в тайлы
//    - addArrowsForNode() - после добавления стрелок в тайлы
//
// Благодаря этим изменениям:
// - При выделении узла он удаляется из тайлов, и пустые тайлы автоматически удаляются
// - При сохранении узла создаются новые тайлы в местах, где теперь находится узел
// - Пустые тайлы, из которых узел ушел, автоматически удаляются
// - Все связи (стрелки), связанные с узлом, также учитываются при обновлении тайлов
